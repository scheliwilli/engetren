# VK бот-тренер ЕГЭ по русскому языку

Бот поддерживает:
- выбор режима: `все`, `подборка N`, `тема <id>`;
- диагностический старт-тест: `диагностика` (15 вопросов);
- персональный маршрут подготовки: `маршрут` (70/20/10);
- бесконечную генерацию заданий по темам ЕГЭ;
- интервальные повторения ошибок (1/3/7/14 дней);
- адаптацию сложности и приоритета тем по ошибкам пользователя;
- хранение прогресса в SQLite (`trainer.db`).

## Темы
- `n_nn` - Н/НН
- `punct` - пунктуация
- `stress` - ударение
- `not_with_word` - НЕ с разными частями речи
- `paronyms` - паронимы

## Установка

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `.env.example` в `.env`.
2. Укажите:
- `VK_BOT_TOKEN` - ключ доступа сообщества;
- `VK_GROUP_ID` - числовой id сообщества.

Пример запуска с переменными окружения в PowerShell:

```powershell
$env:VK_BOT_TOKEN="vk1.a...."
$env:VK_GROUP_ID="123456789"
python bot.py
```

## Команды пользователя
- `начать`
- `темы`
- `тема n_nn` (или другой id темы)
- `все`
- `подборка 10`
- `диагностика`
- `маршрут`
- `статистика`
- `стоп`

## Как работает адаптация
- После каждого ответа бот обновляет персональную статистику по темам.
- В смешанном режиме чаще предлагаются темы с большим количеством ошибок.
- Сложность генерации растёт при высокой точности и снижается при проблемах.

## Важно по безопасности
Если токен сообщества уже был где-то опубликован, его нужно перевыпустить в настройках VK и использовать новый.
